# 🚀 Leanr
### Peer-to-Peer Skill Exchange Platform

**Learn what you lack. Teach what you know.**

Leanr is a peer-to-peer skill exchange platform designed for students who want to learn new skills while sharing their own knowledge with others.

Instead of traditional paid courses, Leanr connects students so they can exchange skills directly and grow together through collaboration.

---

# 🌟 Overview

Leanr creates a collaborative environment where students can discover peers who can teach them a skill while offering their own expertise in return.

The goal is to make learning **accessible, practical, and student-driven**.

Students can:

- Discover skills offered by other students  
- Offer their own knowledge or expertise  
- Connect with peers for skill exchange  
- Build a collaborative learning network

# 💻 Running the Project Locally

### Clone the repository


git clone https://github.com/Deekshitha-089/leanr.git


### Navigate to the project directory


cd leanr


### Install dependencies


npm install


### Start the development server


npm run dev


The application will run at:


http://localhost:5173
